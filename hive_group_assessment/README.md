# HIVE Group 考核工作区

这个文件夹把 Notion 里的 HIVE Group 考核流程整理成了可以直接执行和提交的材料。

## 总目标

在约一个月内打通一条具身智能最小可行学习闭环：

1. 掌握基本的 Python、PyTorch、计算机视觉和机器人概念。
2. 熟悉 Git、Linux、conda、tmux、VSCode Remote、Python 调试等科研常用工具。
3. 学会找文献、读文献、做文献汇报，主线围绕 Diffusion Policy。
4. 复现一个 Diffusion Policy 任务，并把代码、实验记录和结果整理到 GitHub。

## 文件说明

- `01_execution_plan.md`：按周拆分的执行计划和检查清单。
- `02_literature_report_template.md`：文献阅读报告模板。
- `03_diffusion_policy_reproduction_log.md`：Diffusion Policy 复现实验记录模板。
- `04_final_ppt_outline.md`：最终 10-15 页汇报 PPT 提纲。
- `diffusion_policy/README.md`：GitHub 提交用的 Diffusion Policy 复现说明模板。
- `diffusion_policy/run.sh`：服务器实验命令记录脚本模板。

## 建议第一步

先在 GitHub 新建仓库：

```text
embodied-ai-bootcamp
```

然后把本文件夹里有用的内容逐步复制到仓库中。每完成一个阶段，就把代码、笔记、命令记录和截图提交一次，形成清晰的学习与复现记录。
