# 汇报材料

这个文件夹用于存放文献汇报和最终项目汇报。

## 文献汇报

要求：

- 10 页以内。
- 10-15 分钟讲完。
- 内容包括问题背景、方法图、关键细节、实验结果、自己的理解与疑问。

建议文件名：

```text
diffusion_policy_paper_report.pptx
```

## 最终项目汇报

要求：

- 10-15 页。
- 按正式组会标准准备。
- 内容包括具身智能背景、Diffusion Policy 方法、代码结构、复现实验、问题反思和后续计划。

建议文件名：

```text
hive_group_final_report.pptx
```

PPT 提纲可以参考：

```text
hive_group_assessment/04_final_ppt_outline.md
```
