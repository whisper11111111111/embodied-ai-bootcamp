| `mkdir` | 创建文件夹 | `mkdir data` |
| `cp` | 复制文件 | `cp a.txt b.txt` |
| `rm` | 删除文件 | `rm a.txt` |
| `cat` | 查看文本 | `cat README.md` |
| `nvidia-smi` | 查看 GPU 状态 | `nvidia-smi` |
